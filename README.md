# Hello World
아래 명령어로 프로젝트를 만든다  
`zig init-exe`  
  
아래와 같은 파일이 만들어진다  
<pre>
>tree
│  build.zig
│
└─src
       main.zig
</pre>    
      
src/main.zig  
```
const std = @import("std");

pub fn main() !void {
    std.debug.print("Hello, {s}!\n", .{"World"});
}	
```
  
빌드하기    
```
zig build
```      
`zig-out\bin\` 디렉토리에 실행 파일이 만들어진다
